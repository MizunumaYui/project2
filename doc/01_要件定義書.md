# キャラクターEC アプリケーション 要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名
キャラクターECアプリケーション

### 1.2 プロジェクトの目的
キャラクターごとの商品を購入・管理できるECプラットフォームを構築する。

### 1.3 対象ユーザー
- **一般ユーザー（購入者）**: キャラクターグッズを購入したい顧客
- **管理者**: 商品・キャラクター・注文を管理するスタッフ

---

## 2. 機能要件

### 2.1 ユーザー向け機能

#### 2.1.1 ユーザー認証
| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| U-001 | 会員登録 | メールアドレス・パスワードで新規登録 |
| U-002 | ログイン | メールアドレス・パスワードでログイン |
| U-003 | ログアウト | セッションを終了 |
| U-004 | パスワードリセット | メール経由でパスワードを再設定 |
| U-005 | プロフィール編集 | ユーザー情報の更新 |

#### 2.1.2 キャラクター閲覧
| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| C-001 | キャラクター一覧 | 登録されているキャラクターを一覧表示 |
| C-002 | キャラクター詳細 | キャラクターの詳細情報と関連商品を表示 |
| C-003 | お気に入りキャラクター | 気に入ったキャラクターをブックマーク |

#### 2.1.3 商品閲覧・購入
| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| P-001 | 商品一覧 | 全商品またはキャラクター別商品を一覧表示 |
| P-002 | 商品検索 | キーワード・カテゴリ・価格帯で検索 |
| P-003 | 商品詳細 | 商品の詳細情報を表示 |
| P-004 | カートに追加 | 商品をショッピングカートに追加 |
| P-005 | カート管理 | カート内商品の数量変更・削除 |
| P-006 | 購入手続き | 配送先・支払い方法を選択して注文確定 |
| P-007 | 注文履歴 | 過去の注文履歴を確認 |

### 2.2 管理者向け機能

#### 2.2.1 キャラクター管理
| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| AC-001 | キャラクター登録 | 新規キャラクターを登録 |
| AC-002 | キャラクター編集 | キャラクター情報を更新 |
| AC-003 | キャラクター削除 | キャラクターを削除（論理削除） |

#### 2.2.2 商品管理
| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| AP-001 | 商品登録 | 新規商品を登録 |
| AP-002 | 商品編集 | 商品情報を更新 |
| AP-003 | 商品削除 | 商品を削除（論理削除） |
| AP-004 | 在庫管理 | 在庫数の確認・更新 |

#### 2.2.3 注文管理
| 機能ID | 機能名 | 説明 |
|--------|--------|------|
| AO-001 | 注文一覧 | 全注文を一覧表示 |
| AO-002 | 注文詳細 | 注文の詳細情報を確認 |
| AO-003 | 注文ステータス更新 | 発送準備中・発送済み等に更新 |
| AO-004 | 売上レポート | 期間別・キャラクター別の売上集計 |

## 3. 非機能要件

### 3.1 性能要件
- ページ読み込み時間: 3秒以内
- 同時接続ユーザー数: 1,000人以上
- 年間稼働率: 99.5%以上

### 3.2 セキュリティ要件
- HTTPS通信の強制
- パスワードのハッシュ化保存
- SQLインジェクション対策
- XSS対策
- CSRF対策

### 3.3 対応ブラウザ
- Google Chrome（最新版）
- Mozilla Firefox（最新版）
- Microsoft Edge（最新版）
- Safari（最新版）

### 3.4 対応デバイス
- デスクトップPC
- タブレット
- スマートフォン（レスポンシブデザイン）

---

## 4. 用語定義

| 用語 | 説明 |
|------|------|
| キャラクター | 商品が紐づく主体（アニメキャラ、ゲームキャラ等） |
| 商品 | 販売される物品（フィギュア、アクスタ、グッズ等） |
| カート | 購入前に商品を一時保管する機能 |
| 注文 | 購入確定後の取引情報 |
